CSS Tutorial: Positioning

Static Positioning

The position: static; value is the default positioning for all elements. Elements with static positioning are placed in the normal document flow. Their positioning is not affected by the top and left properties.

Static

Relative Positioning

The position: relative; value allows you to adjust an element's position relative to its normal position in the document flow. You can use the top and left properties to move the element from its original position.

Relative

Absolute Positioning

The position: absolute; value removes an element from the normal document flow and positions it relative to its nearest positioned ancestor. You can use the top and left properties to specify its exact position.

Absolute

Fixed Positioning

The position: fixed; value also removes an element from the normal document flow and positions it relative to the viewport (the browser window). Fixed elements remain in the same position even when the page is scrolled.

Fixed
