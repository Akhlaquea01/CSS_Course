CSS Tutorial: Display Property

Block Value

The display: block; value makes an element a block-level element. Block-level elements create a new block in the layout and typically start on a new line. They occupy the full width available by default.

This is a block-level element.

Inline Value

The display: inline; value makes an element an inline-level element. Inline-level elements do not create a new block. They flow within the content horizontally and only take up as much width as necessary for their content.

This is an inline-level element.

Inline-Block Value

The display: inline-block; value combines aspects of both block and inline. Inline-block elements flow within the content horizontally, but they also allow you to set width, height, margins, and padding.

This is an inline-block element. It has a fixed width and height.
