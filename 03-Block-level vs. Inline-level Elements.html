CSS Tutorial: Block vs. Inline Elements

Block-level Elements

Block-level elements create a rectangular block or box in the layout. They typically start on a new line and stack vertically. Some common block-level elements include:

Div (Block)
Paragraph (Block)
Heading (Block)
List (Block)
Table (Block)

Block-level elements occupy the full width available by default and force a line break before and after themselves.

Inline-level Elements

Inline-level elements do not create a new block. They flow within the content horizontally, typically from left to right. Some common inline-level elements include:

Span (Inline) Link (Inline) Strong (Inline) Emphasis (Inline)

Inline-level elements only take up as much width as necessary for their content and do not force line breaks.

Converting Elements
